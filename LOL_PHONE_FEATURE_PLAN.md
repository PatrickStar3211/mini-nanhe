# 手机功能规划

当前阶段已经完成掌盟排位 UI、结算、战绩存档与排位音效；PP 通讯及接单功能仍留待后续实现。

## 掌盟

- 查看当前段位、LP／大区分数、历史最高段位与最近战绩。
- 在掌盟中进行排位；训练页“打游戏”只增加技巧，不影响段位。
- 排位胜利增加 20～25 LP，失败减少 15～20 LP。
- 每层连胜让下一场胜率降低 1%，每层连败让下一场胜率增加 1%，均不限制层数。
- 胜利降低压力，失败增加压力；压力变化随连胜／连败每层增加 2。
- 掌盟首页显示当前段位、历史最高段位与最近战绩，不显示技巧数值。
- 对局找到页显示胜算描述，并强制玩家先接受或拒绝；决定前手机内导航与游戏底部导航均不可离开。
- 接受后立即结算胜负、LP、晋级／掉段与连胜／连败；拒绝不产生战绩。
- 战绩页最多保存并显示最近 10 场，仅展示小南河头像与胜负结果。
- 对局找到、接受、拒绝分别使用独立音效；做出选择时立即中断对局找到音效，等待接受／拒绝音效完整播放后才切换页面。
- 首页、对局确认与结算页使用对应段位徽记；所有卡片、文字和按钮由原生 UI 绘制，美术图仅作为无文字背景与徽记素材。

## PP

- PP 是 NPC 通讯与订单发布入口。
- NPC 通过私讯发送教学、陪玩、代练与灵活组排订单。
- 能否接取只由历史最高段位和技巧决定。
- 接取后的成功率固定 95%，炸单率固定 5%，不影响自己的段位。
- 教学 80／小时；低段位陪玩、代练、灵活 70／小时。
- 大师以上属于高段位：陪玩、代练 100／小时，灵活 120／小时。
- 订单时长、NPC、发布时间与炸单后果留待手机功能实现阶段确定。
